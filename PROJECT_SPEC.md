# 🧠 AI IT 뉴스룸 프로젝트 명세서 (Streamlit + Gemini)

## 📌 프로젝트 개요

국내 IT 뉴스 RSS를 수집하여 AI로 분석하고, 날짜별로 정리된 1장짜리 뉴스 브리핑을 제공하는 웹 애플리케이션을 개발한다.

* 플랫폼: Streamlit Web App
* AI 모델: Gemini API
* 데이터 저장: JSON 파일 (DB 사용하지 않음)
* 배포: Streamlit Cloud
* 저장소: GitHub Repository

---

## 🎯 주요 목표

1. RSS 기반 뉴스 자동 수집
2. 최근 뉴스 AI 분석 및 요약
3. 토픽별 뉴스 그룹화
4. 날짜별 1페이지 뉴스 브리핑 생성
5. 관리자 대시보드 제공
6. 방문자 통계 제공

---

## 🏗️ 시스템 구조

```
[RSS 피드]
   ↓
[뉴스 수집 (Python)]
   ↓
[JSON 저장]
   ↓
[Gemini AI 분석]
   ↓
[토픽 그룹화 + 리포트 생성]
   ↓
[Streamlit UI 출력]
```

---

## 📁 프로젝트 구조

```
ai-newsroom/
│
├── app.py                  # 메인 뉴스 화면
├── admin.py                # 관리자 대시보드
├── rss_collector.py        # RSS 수집 모듈
├── ai_analyzer.py          # AI 분석 모듈
│
├── data/
│   ├── feeds.json          # RSS 목록
│   ├── raw_news.json       # 수집 뉴스
│   ├── visitors.json       # 방문자 로그
│   └── reports/
│       ├── 2026-03-19.json
│       └── ...
│
├── requirements.txt
└── .streamlit/
    └── secrets.toml
```

## 🧩 기능 상세 명세

### 1️⃣ RSS 관리 기능 (관리자)

* RSS URL 추가
* RSS URL 삭제
* 현재 RSS 목록 확인

---

### 2️⃣ 뉴스 수집 기능

* 등록된 RSS 피드에서 뉴스 수집
* 다양한 RSS 포맷 대응 (feedparser + fallback 파싱)
* 최소 최근 데이터 확보 (날짜 오류 허용)

---

### 3️⃣ AI 분석 기능

* 모델: Gemini API 사용
* 뉴스 리스트 전체를 한 번에 입력
* 수행 작업:

  * 뉴스 요약
  * 토픽별 그룹화
  * 중요 뉴스 선별
  * 링크 포함

---

### 4️⃣ 리포트 생성

출력 형식:

```
📌 오늘의 IT 뉴스 브리핑 (YYYY-MM-DD)

[토픽 1]
- 핵심 요약
- 주요 뉴스 링크

[토픽 2]
- 핵심 요약
- 주요 뉴스 링크
```

---

### 5️⃣ 데이터 저장 방식

* 모든 데이터는 JSON 파일로 저장
* DB 사용 금지

#### 예시

```json
{
  "date": "2026-03-19",
  "content": "AI가 생성한 뉴스 브리핑 텍스트"
}
```

---

### 6️⃣ 메인 화면 (app.py)

* 최신 날짜 뉴스 자동 표시
* 날짜별 네비게이션 (사이드바)
* 리포트 내용 출력

---

### 7️⃣ 관리자 대시보드 (admin.py)

#### 기능

* RSS 관리
* 수집 실행
* AI 분석 실행
* 통합 실행 버튼
* 방문자 수 표시

---

### 8️⃣ 방문자 통계

* 접속 시 visitors.json에 기록
* 총 방문자 수 표시

---
